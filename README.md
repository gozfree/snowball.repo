# snowball.repo
This is a script simplified the git command for each folder:
  * filewatcher
  * git_repo
  * ipcam
  * libraries
  * rasberrypi
  * README.md
  * rpcd
  * sktool
  * useful_scripts
  * vimrc


## How To Use
  * like git command

  $ ./git_repo clone

  $ ./git_repo pull

  $ ./git_repo push

  $ ./git_repo log

  $ ./git_repo status
